PyCon APAC 2014

PyCon APAC (Asia Pacific) は、Pythonコミュニティによって、Pythonコミュニティのために開催されるカンファレンスです。PyCon APACは毎年一度開催され、初心者からコアデベロッパーまで様々なレベルのユーザに楽しんでいただけます。もともとの PyCon は 2003 年に北米で始まりましたが、今では世界中で PyCon の精神を受け継いだカンファレンスが開催されています。

全ての参加者とコミュニティによりよくPyConを楽しんでいただくため、今年は Code of Conduct を設定しました。ぜひご一読いただき、一緒にすばらしいカンファレンスにしましょう。

Background

PyCon APAC/TW 2014 は、アジア太平洋地域の Pythonista によって企画運営されています。今回は台湾の台北市で開催します。この一大イベントに、ぜひご参加いただき、あるいはスポンサーとしてご協賛いただき、ブランドや製品のご紹介を通して、御社の事業にお役立てください。 PyCon APAC/TW 2014 の実行組織は、ほとんどが地元の Python ユーザグループのメンバーで、 PyCon TW (2012 と 2013) の企画運営も務めました:

  • Python Hsinchu User Group (PyHUG)
  • Taipei Python User Group (Taipei.py)
  • Python Tainan User Group (Tainan.py)
  • PyLadies Taiwan
  • Hualien Python User Group (Hualien.py)

PyCon APAC は国際的なイベントです。最初の APAC はシンガポールで 2010 に開催されました。その後、 2013 年の東京開催を経て、今年 2014 年は台湾の台北でかいさいされることとなりました。APACの開催のため、今年は PyCon Taiwan 2014 としての開催はありません (実質的には、PyCon APAC 2014 が PyCon Taiwan 2014 を兼ねています。)
