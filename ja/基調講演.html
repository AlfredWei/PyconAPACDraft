Stephen McDonald

Stephen McDonaldはソフトウェアプログラマーです。1990末期、Nescape GoldとICQをはやっていた時からネットワークに関するソフトウェアの開発をし始めました。

Stephenさんは数多くのPythonとDjangoのベンチャー企業をかかわることがあります。普段では、Mathspaceの技術管理者として働いており、暗記法技術を応用して数学教育の改革を行っています。他の時間は新聞閲覧ソフトKouioを使い、数万件程度の世界ニュースを分析します。

StephenさんはMezzanineを開発・管理すます。MezzanineはDjangoを基づいて、人気のあるCMSプロジェクトです。

 

ジェシカ・マッケラー

ジェシカ・マッケラーは、アメリカ合衆国マサチューセッツ州ケンブリッジに住む起業家であり、ソフトウェアエンジニアであり、オープンソース開発者です。Python Software Foundationのディレクターをつとめ、世界最大のPythonユーザグループを運営しています。そのユーザグループとBoston Python Workshopというイベントを開催して、何百人もの女性を地元Pythonコミュニティに紹介することに成功し、その後同じようなプログラムがアメリカ全土でが開催されるまでになりました。ジェシカは経験豊富なオープンソースソフトウェア開発者でもあり、いくつかのオープンソースプロジェクトを管理しています。その中にはOpenHatchやイベント駆動ネットワークエンジンのTwistedも含まれ、「The Architecture of Open Source Applications Volume II and the second edition of O’Reilly’s Twisted Networking Essentials」という書籍ではTwistedの章も著しています。

 

Wes McKinney

Wes　McKinneyはPython hacker、作家および起業家です。彼はpandasの作者で、O’Reilly社からPython for Data Analysisという本が出版されました。彼もDataPadを創設して、CEOを営みます。DataPadはサンフランシスコに創設し、分析とビジネスロジックのことを注目します。Mckinney氏はstatsmodelsプロジェクトのプログラムコード開発の一員です。AQR Capital Managementに仕事していたとき、Pythonでビジネスロジックとデータサイエンスのデータ処理の割合が多いので、pandasというツールを開発しました。

 

フェルナンド・ペレス

フェルナンド・ペレスはカリフォルニア大学バークレー校の研究科学者であり、オープンなツールを使用した高度な科学技術計算と、脳内イメージ化の分野で活躍しています。コロラド大学ボルダー校の物理学修士課程在学中の2001年にIPythonプロジェクトを始めました。その後も有能な（そしてよく働く）チームを率いて、キーボードを叩く人間と機械の中のビットデータの間のインターフェイスとしてのIPythonの開発を続けています。

彼はNumFOCUSの創業メンバーであり、PSFメンバーであり、2012年には「Award for the Advancement of Free Software」をIPythonプロジェクトとPythonの科学分野での発展に貢献したとして受賞しました。また、2013年に設立されたBerkeley Institute for Data Scienceの創業研究者でもあります。

コンピュータ以外では、山をハイキングをしたり、バックパックやクライミングなどを満喫しています。より詳しく知りたい方はhttp://fperez.orgをご覧ください。

 

Andreas Klöckner

Andeas Klöcknerはイリノイ大学Urbana-Champaign校情報科学専攻・科学計算研究の助教授です。彼の研究はwaveシミュレーションのhigh-order方法及び高性能大規模の並列科学計算です。自身の研究の為、Klöckner氏は数多く科学計算用のPythonソフトを開発しました。その中に、PyOpenCL、PyCUDA、PuDB Python debuggerを含みます。Andeas Klöcknerはブラウン大学応用数学専攻に、Jan Hesthavenに指導され、博士学位を取りました。研究は大規模の有限要素模擬です。その後、ニューヨーク大学に教師になり、Leslie Greengrad’s研究グループを参加しました。研究は積分方程式及び高速計算法です。
