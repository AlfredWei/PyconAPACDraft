Jessica McKellar

Jessica McKellar 是來自美國麻州劍橋的企業家、軟體工程師、及開源開發者。她是 Python Software Foundation 的現任董事，也是波士頓 Python 同好會的組織幹部，這是世界上最大的 Python 使用者團體。她所籌辦的波士頓 Python 工作坊，透過循序漸進的系列課程，協助數以百計的女性加入當地的 Python 社群，而這個成功經驗也在美國各地被模仿複製。 Jessica 是資深的開源貢獻者，負責維護多項開源專案，例如 OpenHatch 及 Twisted event-driven network engine 等。她也撰寫技術文件和書籍，像是 The Architecture of Open Source Applications Volume II 裡的 Twisted 專章，還有 O’Reilly 出版的 Twisted Networking Essentials 第二版。

Fernando Pérez

Fernando Pérez 目前是 UC Berkeley 的研究員，研究結合使用開放工具的高階科學計算及腦部影像。他在 2001年就讀 CU Boulder 物理研究所時， 創建了 IPython 計畫。他帶領一個優秀（且努力）的團隊，持續開發 IPython，橋接電腦後的數字與鍵盤前的人。

他是 NumFOCUS 的創始會員與 PSF 會員。由於  IPython 及對於科學 Python 的貢獻，得到了 2012 年 Advancement of Free Software 獎項。 他也是在 2013 年創建的 Berkeley Institute for Data Science 的創始研究員。

當不坐在電腦前時，Fernando 嚮往健行及登山。 請到 http://fperez.org 查看更多資訊。

Wes McKinney

Wes McKinney 是開源的 Python hacker、作家、與企業家。他是 pandas 的作者並由 O'Reilly 出版了 Python for Data Analysis 一書。他是 DataPad 的創辦人與 CEO。該公司位於舊金山，從事分析與業務邏輯 (business logic) 工作。偶爾他也會貢獻程式碼到 statsmodels 專案。他在 AQR Capital Management 工作時，發現愈來愈需要透過 Python 來處理業務邏輯與資料科學 (data science) 問題，因此開發了 pandas 這套工具。

Andreas Klöckner

Andreas Klöckner 是伊利諾大學 Urbana 分校電腦科學系， 科學計算研究群的助理教授．他的研究包括 wave 模擬的 high-order 數值方法以及高效能大規模的平行處理的科學計算. 為了支持他的研究 Dr. Klöckner 發展了許多科學計算用的 Python 軟體組件，包括 PyOpenCL、PyCUDA、PuDB Python debugger。Andreas Klöckner 在布朗大學應用數學跟隨 Jan Hesthaven 取得博士學位，主要研究的是大尺度的有限元素模擬，之後到紐約大學擔任講師與加入 Leslie Greengard's 研究群，主要研究積分方程以及高速的演算法。
