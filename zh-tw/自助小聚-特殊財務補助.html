國外參與者財務補助計畫

我們誠摯邀請國外 Python 愛好者參與 PyCon APAC 2014，促進會眾的優質交流。但對於國外參與者而言，路途遙遠，旅費昂貴，是以在一般的財務補助方案之外，我們規劃了國外參與者特別財務協助方案 (Special Financial Aid Plan for Foreign Pythonista )，希望協助對會議有興趣，但資金支出較為困難的 Python 愛好者們參與這次會議。

由於能夠補助的數量有限，這種情況下我們希望受理的對象為：「能夠主持 BoF 的會議參與者」。

  • 本方案採審核制
  • 我們會以BoF主題與 Python 相關程度，包含相關議題或套件的知名度與主題之特殊性來做篩選。
  • 隨到隨審。
  • 篩選是以 BoF 提案者為單位。
  • 主持 BoF 的方式可自由發揮，唯活動內容需遵守 Code of Conduct 及能正向促進社群發展
  • BoF 活動如果對場地或是器材有所需求，可於申請時提出，我們會設法協助

申請 Financial Aid 的方式如下：

  • 請依循正常管道至 KKTIX 網址完成 BoF 報名。 報名內容為：
    • 國籍
    • BoF 文案
    • 負責人姓名、聯絡方式 (包括 email 與電話)
  • Financial Aid 受理時間為即日起至 BoF 申請截止日(4/30)。

我們會在五月 1 日前公告合格 BoF 提案、並電郵通知各負責人審核結果。若通過申請，我們將聯絡負責人，於BoF舉辦時期給予現金款項.

我們知道每個人的經濟狀況，或者是對 Python群體的貢獻是難以比較的。而 Financial Aid 的預算有限，無法保證能幫到所有人，但我們還是想對有困難的朋友們說：「需要幫忙就說一聲吧！我們盡力！」

任何進一步的問題，請寄信至 registration[at]pycon.tw
